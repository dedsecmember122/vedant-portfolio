# Vedant Sharma — Portfolio

## 🚀 Deploy to Vercel (3 steps)

1. Go to https://vercel.com → Sign up free with GitHub
2. Click "Add New Project" → "Upload" → drag this entire folder
3. Click Deploy — done in 60 seconds!

## 📹 Add Your ANPR Video

After uploading your video to YouTube (unlisted):
1. Open index.html
2. Find: `var YT_VIDEO_ID = 'PASTE_YOUR_YOUTUBE_ID_HERE';`
3. Replace with your YouTube video ID (the part after v= in the URL)
   Example: https://youtube.com/watch?v=ABC123def → ID is `ABC123def`
4. Save and redeploy

## 🌐 Custom Domain (vedantsharma.com)

1. Buy domain at namecheap.com or hostinger.in
2. In Vercel → Project Settings → Domains → Add your domain
3. Follow DNS instructions shown by Vercel
